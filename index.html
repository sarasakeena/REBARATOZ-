What REBARATOZ Does

A comprehensive RCC detailing automation engine — see exactly what goes in and what comes out for each element type.

01

Slab Reinforcement

RCC Slab
Input Parameters

Slab span, thickness, support condition (simply supported / continuous), steel grade & bar spacing from design report

Span × Width Slab Thickness Top & Bottom Bars Crank %
Output Drawing

Full plan layout with top & bottom rebar grid, crank bar elevation detail, distribution steel, dimensions, and bar schedule table — in editable DXF

T12 @ 150 c/c TOP T10 @ 200 c/c BTM DXF Ready
02

Beam Reinforcement

RCC Beam
Input Parameters

Beam span, width & depth, top & bottom longitudinal bars, stirrup diameter & zone spacing from design output

Beam Span b × D Top / Btm Bars Stirrup Zones
Output Drawing

Longitudinal elevation with curtailment zones, cross-section detailing at support & midspan, stirrup density map — all dimensioned

3T20 TOP CONT. T8 @ 100/200 c/c DXF Ready
03

Column Reinforcement

RCC Column
Input Parameters

Column size, floor height, main bar count & diameter, lateral tie diameter & pitch, lap splice zone from design schedule

Col. Size Floor Height Main Bars Ties Pitch
Output Drawing

Plan section showing bar arrangement & tie pattern, elevation with lap splice zone, floor levels & coordinate grid placement

8T20 + T8@150 Lap Zone 50d DXF Ready
04

Staircase Reinforcement

RCC Stair
Input Parameters

Riser height, tread width, number of steps, waist slab thickness, main bar & distribution bar specs, stair type (dog-leg / open-well)

Riser × Tread Waist Thickness Stair Type
Output Drawing

Flight profile with waist slab detailing, landing bars, anchorage lengths at supports, tread/riser dim lines & bar mark schedule

T12 @ 120 MAIN T8 DIST. BARS DXF Ready
